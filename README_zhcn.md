## 简介

<img src="https://ae01.alicdn.com/kf/He80bcdd2ecc64a0890480c5848fee12bW.png">

SPlayer 是一款HTML5方块形音乐播放器，支持常见的音乐文件类型（mp3,wav,flac等）

## 特点

 - 小巧：方块形，占位小。
 - 易用：对接网易云音乐，直接获取音乐。
